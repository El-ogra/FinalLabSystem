# F-21 — Specialized Test TPT Decision

## Status: Deferred

Table-per-type (TPT) inheritance for specialized test types
(Blood Bank, Andrology, Culture) was evaluated and deferred
pending a concrete business requirement.

The current single-table model (TestType) is sufficient for
the existing feature set. Revisit when a new specialized
test type requires fields not representable in the current
schema.

**Deferred date:** June 2026
**Revisit trigger:** New specialized test type requirement
